== Getting Started

1. Add <tt>gem 'rails_print_template' </tt> to your Gemfile.
2. Run <tt>bundle install</tt>
3. Run <tt>rake rails_print_template:install:migrations</tt> inside your Rails application
4. Run <tt>rake db:migrate</tt>
